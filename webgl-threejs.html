Graphics on your browser

What is WebGL?

A graphics library for your browser where you can render low-level 3D and 2D graphics on the HTML5 Canvas.

WebGL Features and Properties

  • Based on OpenGL ES 2.0.
  • It uses a Shader-based API for textures via GLSL(OpenGL shading Language).
  • Image Processing.
  • 3D physics simulation.
  • And so much more ...
